EDMA Documentation

Most EDMA information is written in Spanish. We are working on translating it to English, and any help with this will be welcome. Russell Reed has begun working on translating the EDMA documentation, but further help is still needed. You can contact  David Martínez Oliveira or Russell Reed for working on translating the documentation.

Source code is mostly commented in English, and this is the main information resource available at the moment, since the example programs are very simple.

Some tutorials have been written and will be translated to English when we can.
